## StepByStep-Redux

Redux入门系列，通过这几篇文章的描述，会对Redux中的一些核心概念有个比较清楚的认识。

最后介绍了react-redux库是如何工作的，以及Provider和Connect模块的作用。

每篇文章的结尾都会有一些简单的demo，帮助理解文章中介绍的内容。
